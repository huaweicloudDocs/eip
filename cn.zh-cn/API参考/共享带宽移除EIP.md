# 共享带宽移除EIP<a name="eip_apisharedbandwidth_0005"></a>

## 功能介绍

共享带宽移除EIP。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=EIP&api=RemovePublicipsFromSharedBandwidth)中调试该接口。

## URI

POST /v2.0/\{project\_id\}/bandwidths/\{bandwidth\_id\}/remove
