# 查询弹性IP项目标签<a name="eip_apitag_0006"></a>

## 功能介绍

查询租户在指定区域和实例类型的所有标签集合。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=EIP&api=ListPublicipTags)中调试该接口。

## URI
