# 更新EIP<a name="eip_api_0004"></a>

## 功能介绍

更新EIP，将EIP跟一个网卡绑定或者解绑定，转换IP地址版本类型。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=EIP&api=UpdatePublicip)中调试该接口。

## URI

PUT /v1/\{project\_id\}/publicips/\{publicip\_id\}
